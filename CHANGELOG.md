# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of Bank Customer Churn Analysis platform
- Random Forest ML model for churn prediction
- REST API with prediction, dashboard, and translation endpoints
- Interactive web interface with Chart.js visualizations
- Multi-language support (English, Hindi, Telugu)
- Dashboard with KPI cards and analytics charts
- Feature importance analysis
- Responsive dark-mode design
- Docker containerization support
- CI/CD pipeline configuration
- Comprehensive test suite
- Documentation (README, CONTRIBUTING, USER_MANUAL, AGENTS)

### Technical
- Flask 3.0 backend with scikit-learn ML pipeline
- Synthetic data generation for 5000 customer records
- joblib model serialization
- i18n with server-side translation dictionaries
- Modern CSS with custom properties and responsive breakpoints
